# pddTrackingExport
**拼多多订单快递查询与 Excel 导出脚本**  A Tampermonkey userscript that fetches your **Pinduoduo delivery information** (tracking number, pickup code, item name, etc.) directly via internal API — then exports it to Excel or copies it to clipboard.   一个用于在拼多多网页端一键获取并导出快递信息的油猴脚本。
